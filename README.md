# 🔢 AI Handwritten Digit Recognizer

This project was developed as part of the KODBUD AI Internship.

## 📌 Project Overview

This application recognizes handwritten digits (0–9) using a Deep Learning model trained on the MNIST dataset.

## 🛠️ Technologies Used

- Python
- Google Colab
- TensorFlow
- Keras
- MNIST Dataset
- Deep Learning

## ✨ Features

- Handwritten digit recognition
- Deep Learning implementation
- MNIST dataset training
- Digit prediction
- Model accuracy evaluation

## 🚀 How to Run

1. Open the notebook in Google Colab.
2. Run all cells.
3. Train the model.
4. View the predicted digit and accuracy.
